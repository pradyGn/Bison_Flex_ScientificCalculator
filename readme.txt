- Use paranthesis while using sin, cos, tan, log2, log10, sqrt, abs, ceil, floor and factorial functions.

Example: sin(x), tan(PI/2), log2(expr), sqrt(25), ceil(1.08), floor(22.99) and (5)!

- do not use paranthesis while using currency conversion or distance conversion. 
- Unlike the provided documents syntax, expression / numbers to be converted to anouther currency or distance should be mentioned after the keyowrd. 

Example: km_to_mi 23, usd_to_gbp 10.
